  1. Suppose I have the following Java code: 
    public class MyApp extends JFrame
{
  /* more code here */    
}
    What does this code do? (4 points)

    This code defines a new class called MyApp which is a subclass of JFrame

  2. Most object oriented programming languages provide inheritance. What are the benefits of this feature? (4 points)

    There are several good answers:

    • Allows code to be reused without copying it to a new module
    • Provides a way to represent specialized cases of a general class
    • Provides a way to capture the common data and behavior of multiple specific classes

    (Other answers might also be acceptable.)
