  1. Java uses interfaces to provide capabilities that might be implemented using multiple inheritance in other programming languages (for instance, in C++). Java interfaces are somewhat similar to abstract classes. In the space below, list two differences between interfaces and abstract classes. (4 points) 
    There are several good answers:

1) Abstract classes can define member data; interfaces cannot
2) Abstract classes can include code for some concrete methods, as long as they
   have at least one abstract method which has only a declaration.
   In an interface, only declarations of methods are allowed.
3) A class can extend only one abstract class, but can implement an
   unlimited number of interfaces.
4) The concept of a static method is meaningful for an abstract class,
   but not for an interface.
  2. Which of the following statements are true about UML (Unified Modeling Language)? Circle all true statements. (3 points, no partial credit)

    1. UML was created by IBM.

    3. UML is only useful for object-oriented software systems.
    4. UML focuses on the relationships between classes, objects or modules, not the detailed logic of individual algorithms or methods.
    5. UML can be applied to the design of web applications.
    6. UML is an official standard controlled by the Object Management Group (OMG)
    7. None of the above statements is true.
